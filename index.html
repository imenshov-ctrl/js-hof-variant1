// Data Transformation Library · Variant 1
1 — map / filter / reduce
map pure · curried

Власна реалізація map через for-цикл. Приймає функцію трансформації, повертає нову функцію, що приймає масив.

const double = map(x => x * 2); double([1, 2, 3, 4, 5]); // → [2, 4, 6, 8, 10] map(x => x.toUpperCase())(['a','b','c']); // → ['A', 'B', 'C']
filter pure · curried

Залишає лише ті елементи, для яких предикат повертає true. Оригінальний масив не змінюється.

const isEven = filter(x => x % 2 === 0); isEven([1, 2, 3, 4, 5, 6]); // → [2, 4, 6] filter(x => x.age >= 18)([ { name:'Alice', age:17 }, { name:'Bob', age:22 }, ]); // → [{ name:'Bob', age:22 }]
reduce pure · curried

Згортає масив до одного значення. Підтримує довільний початковий акумулятор, зокрема об'єкти та масиви.

const sum = reduce((acc, x) => acc + x, 0); sum([1, 2, 3, 4, 5]); // → 15 // частотна карта слів reduce((acc, w) => ({ ...acc, [w]: (acc[w]||0)+1 }), {}) (['a','b','a','c','b','a']); // → { a:3, b:2, c:1 }
2 — compose та pipe
compose right-to-left

compose(f, g, h)(x) === f(g(h(x))). Математична нотація — функції застосовуються справа наліво.

const process = compose( x => x.toUpperCase(), // 3. → 'HELLO WORLD' x => x.trim(), // 2. → 'hello world' x => x.replace(/,/g,'') // 1. → 'hello, world' → ... ); process(' hello, world '); // → 'HELLO WORLD'
pipe left-to-right

pipe(f, g, h)(x) === h(g(f(x))). Зручна для читання: дані «течуть» зліва направо крізь трансформації.

const normalize = pipe( s => s.toLowerCase(), s => s.trim(), s => s.replace(/\s+/g, '-'), s => `slug:${s}` ); normalize(' Hello World '); // → 'slug:hello-world'
3 — curry
curry N-arity → chain

Перетворює функцію від N аргументів на ланцюжок функцій від одного аргумента. Підтримує довільну кількість аргументів.

const add = curry((a, b, c) => a + b + c); add(1)(2)(3); // → 6 add(1, 2)(3); // → 6 add(1)(2, 3); // → 6 add(1, 2, 3); // → 6 const add10 = add(10); const add10to5 = add10(5); add10to5(3); // → 18
partial часткове застосування

Фіксує перші аргументи функції, повертаючи нову функцію, що очікує решту. Корисно для створення спеціалізованих версій загальних функцій.

const multiply = (a, b) => a * b; const triple = partial(multiply, 3); triple(5); // → 15 triple(10); // → 30 const greet = (greeting, name) => `${greeting}, ${name}!`; const hello = partial(greet, 'Hello'); hello('Михайло'); // → 'Hello, Михайло!'
5 — memoize
memoize кеш · benchmark

Кешує результати виклику чистої функції. Повторний виклик з тими самими аргументами повертає збережений результат миттєво, без повторного обчислення.

// Число Фібоначчі (без мемоізації — експоненційний час) const fib = memoize(n => n <= 1 ? n : fib(n - 1) + fib(n - 2) );
Chainable API
chain(data).map().filter().reduce() fluent interface

Обгортка Chain дозволяє будувати ланцюжки трансформацій у читабельному стилі. Оригінальний масив не змінюється — кожна операція створює нову Chain.
